# Ôn tập Kế toán – Thuế · TOPA

Website học tập trực quan cho thực tập sinh **Kế toán – Thuế**. Truy cập qua GitHub Pages (không cần cài gì).

## Nội dung tương tác
- **📊 Sơ đồ & Infographic** — 9 chủ đề (khấu trừ GTGT, hóa đơn, chi phí được trừ, timeline tờ khai, mức phạt, TNCN, thuế suất, BHXH…), mỗi chủ đề có sơ đồ luồng + infographic, xem phóng to.
- **🧠 Mindmap** — bản đồ tư duy tổng quan.
- **🎴 Flashcards** — 75 thẻ Hỏi–Đáp, lật thẻ, lọc theo chủ đề, chế độ học từng thẻ.
- **✍️ Trắc nghiệm** — 28 câu luyện tập: chọn đáp án → biết đúng/sai ngay + giải thích căn cứ pháp luật + chấm điểm.

## Kỹ thuật
- Web tĩnh thuần (HTML/CSS/JS), không phụ thuộc thư viện ngoài. Dữ liệu ở `data/*.json`.
- Ảnh sơ đồ ở gốc repo (`so-do-*.png`), infographic tối ưu web ở `img/`.
- Responsive (điện thoại/máy tính), có giao diện sáng/tối.

## ⚠️ Lưu ý
- Phần **Trắc nghiệm** là để **LUYỆN TẬP** — bài kiểm tra chính thức (Module 06) do mentor ra đề riêng.
- Nội dung câu hỏi/đáp án **do AI trích xuất** từ tài liệu NotebookLM (suy từ VBPL 2025–2026) — **mentor rà soát** trước khi dùng chính thức.
